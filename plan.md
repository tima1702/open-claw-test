# OpenClaw Landing Page — Plan

## О проекте OpenClaw
OpenClaw — open-source AI-агент (250k+ звёзд на GitHub).
Персональный ассистент: почта, календарь, браузер, файлы через мессенджеры.
Поддерживает Claude, GPT, DeepSeek, локальные модели. macOS/Windows/Linux.

## Цель
Лендинг-лид-магнит: бесплатный PDF-гайд по настройке OpenClaw в обмен на подписку в Telegram.
Проверяем спрос, собираем аудиторию, прогреваем на платные услуги.

## Воронка

```
Лендинг (PDF-гайд бесплатно)
    -> Подписка на Telegram-канал/бот -> получает PDF
        -> Человек пробует настроить сам по гайду
            -> Пишет в чат: получилось / не получилось
                -> Помогаем, общаемся
                    -> Апсейл: платная поддержка / настройка под ключ / автоматизация бизнеса
```

## Оффер на лендинге
Бесплатный PDF-гайд "Как настроить OpenClaw с нуля" — пошаговая инструкция.
Для получения — подписаться на Telegram.

## Платные услуги (не на лендинге, продаём через Telegram)
- Помощь с настройкой если по гайду не получилось
- Настройка VDS под ключ
- Персональное сопровождение и поддержка
- Автоматизация бизнес-процессов через OpenClaw

## Целевая аудитория
- Предприниматели — хотят AI-ассистента для бизнеса
- Фрилансеры — автоматизация рутины
- Менеджеры/маркетологи — видели хайп, хотят попробовать
- Начинающие разработчики — не уверены в DevOps-части
- Люди без IT-бэкграунда — хотят "свой приватный ChatGPT"

## Боли ЦА
- "Хочу OpenClaw, но не понимаю с чего начать"
- "Потратил выходные — ничего не работает"
- "Docker, Node.js, конфиги — это не мой язык"
- "Настроил, но мессенджер не подключается"
- "Боюсь что-то сломать или слить API-ключи"

## Скоп работы

### 1. Структура лендинга
- **Hero**: заголовок + подзаголовок + CTA (получить гайд)
- **Боли**: "Знакомо?" — 3-4 пункта
- **Что внутри гайда**: содержание PDF
- **Как получить**: подпишись на Telegram -> получи PDF
- **CTA повторный**: кнопка Telegram внизу
- **Footer**: минимальный

### 2. Контент лендинга

**Заголовок (варианты):**
1. "Настройте свой AI-ассистент за вечер — бесплатный гайд по OpenClaw"
2. "OpenClaw за 2 часа: бесплатный пошаговый гайд"
3. "Хотите свой AI-ассистент? Начните с бесплатного гайда"

**Подзаголовок:**
"Пошаговая инструкция по настройке OpenClaw — от нуля до работающего AI-ассистента в вашем Telegram. Без DevOps-опыта."

**Блок "Знакомо?":**
- Видите как все обсуждают OpenClaw, но не понимаете с чего начать
- Пробовали настроить — застряли на Docker и конфигах
- Хотите AI-ассистента, но не хотите тратить неделю на настройку
- Читали документацию — стало только сложнее

**Блок "Что внутри гайда":**
- Пошаговая установка на VDS (без Docker-боли)
- Подключение Telegram / WhatsApp
- Безопасная настройка API-ключей
- Оптимизация расхода токенов
- Чек-лист "всё работает"

**CTA:** "Получить гайд в Telegram" (ссылка на TG канал/бот)

### 3. Технологии
- **HTML + CSS + JS** — чистая статика
- `index.html` + `style.css` + `script.js`
- CTA-кнопка — ссылка на Telegram (t.me/...)
- Трекинг — JS (клики по CTA, скролл, время на странице)
- Формы нет — вместо неё кнопка Telegram
- Деплой — ручной

### 4. Сбор данных
- **Лиды**: через Telegram (подписка = лид)
- **Трекинг на лендинге**: клики CTA, скролл до блоков, время на странице
- Аналитика — простой JS в localStorage или console.log (MVP)
